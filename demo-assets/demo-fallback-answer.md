# AI Fallback Answer

Use this if the live model provider is slow or unavailable during the demo.

## 中文解释

这段内容说明 VibeReader 的核心价值不是单独替代 PDF 阅读器或聊天窗口，而是把阅读、选区、提问和批注放在同一个工作台里。用户不需要在文档、笔记和 AI 工具之间来回切换。

## 三点总结

- 左侧文档保持可见，减少提问时丢失上下文的问题。
- 右侧 AI 面板承接选区内容，可以继续总结、追问或生成复习材料。
- 高亮和笔记先保存为本地阅读记录，保证没有 AI 时仍能完成阅读整理。
