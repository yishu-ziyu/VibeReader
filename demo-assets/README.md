# VibeReader Demo Assets

These files are the stable local assets for Hackathon demo rehearsal.

- `outline-demo.pdf`: small PDF with three bookmarks for outline jump testing.
- `wonderland_short.pdf`: real multi-page PDF for rendering, paging, zooming, and text selection.
- `sample.md`: Markdown article for the universal reader path.
- `sample.txt`: Plain-text note for the universal reader path.
- `sample.html`: HTML article used to verify safe text extraction.
- `demo-fallback-answer.md`: prewritten answer for live AI outage fallback.

Keep this directory self-contained. Demo scripts should not depend on `/tmp` files or the old Zotero fork.
